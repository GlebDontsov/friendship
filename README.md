# friendship
Это API для магазина, которое позволяет управлять товарами, заказами, пользователями и другими элементами вашего магазина.

# Запуск backend
```docker-compose build```

```docker-compose up```

# Технологии
- Python
- Django REST Framework
- PostgreSQL
- Docker
- Docker Compose
- Celery
- MailHog
- Redis
- OAuth2
